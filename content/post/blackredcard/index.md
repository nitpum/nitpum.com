อันนี้ลองทำเว็บนับใบดำใบแดงดู จะได้ช่วยคำนวณโอกาสที่จะใบไหนได้ ใครต้องไปจับหรือไปเชียร์ญาติลองเอาไปใช้ดูได้ครับ [blackredcard.nitpum.com](https://blackredcard.nitpum.com/)

{{<notice note>}}
จริงๆ อันนี้เคยเอาไปโพสต์ที่อื่นมาแล้ว แต่รอบมาโพสต์ในแพลตฟอร์มของตัวเองบ้างจะได้ไม่หายไปไหน
{{</notice>}}

## เบื้องหลัง
- เขียนด้วย HTML ไฟล์เดียว
- ไม่ได้ใช้ AI ช่วยเลย ทำเองทั้งหมด (ใช้ในงานเยอะแล้ว อยากกลับมาเขียนอะไรเองบ้าง)
- ลองเล่น [alpine.js](https://alpinejs.dev/) อยากลองมานานแล้ว เขียนสนุกดี
- หลักๆ logic เขียนไม่นาน เสียเวลาส่วนใหญ่ไปดีไซน์หน้าเว็บมากกว่า
- deploy ไว้บน Cloudflare Pages ไม่ต้อง host เองแถมได้พลัง CDN ด้วย
