สวัสดีครับพอดีช่วงที่ผ่านมาไม่นานนี้มีโอกาสได้ลองย้ายแอพธนาคารไปมือถืออีกเครื่อง โพสต์นี้เลยจะมาบันทึกไว้ว่าในปี 2026 นี้ แอพธนาคารไทยแต่ละเจ้าย้ายไปอีกเครื่องต้องใช้อะไรบ้างแล้วย้ายง่ายแค่ไหน 

ทุกแอพผมใช้งานผ่านเน็ตซิมมือถือทั้งหมดไม่ได้ต่อ Wi-Fi เลยไม่ได้ทดสอบว่าแอพไหนต่อ Wi-Fi ใช้งานได้บ้าง แต่เข้าใจว่าส่วนใหญ่น่าจะไม่อนุญาตให้ใช้ดังนั้นเครื่องใหม่ที่เราจะย้ายควรใส่ซิมที่ใช้เน็ตได้ไว้นะครับ

ด้านล่างนี้เป็นตารางสรุปครับ

| แอพธนาคาร  | เลขบัตรประชาชน | เลขบัญชี | เบอร์โทรศัพท์มือถือ | ยืนยัน OTP SMS | แสกนหน้า KYC | หมายเหตุ |
| ----- | ----------- | ----- | ----------- | --------------- | ----- | ----- |
| SCB EASY | ✅ | ✅ | ✅ | ✅ | ✅ | - |
| K PLUS | ✅ | ✅ | ✅ | ✅ | ✅ | - |
| krungsri | ✅ | ✅ | ✅ | ✅ | ✅ | ต้องออกจากระบบในเครื่องเดิมก่อนไม่งั้นจะต้องไปทำที่สาขาพร้อมบัตรประชาชน |
| Krungthai NEXT | ✅ | - | ✅ | ✅ | ✅ |  - |
| ttb touch | ✅ | - | - | ✅ | ✅ | - |
| เป๋าตัง | ✅ | - | ✅ | ✅ | 🟨 | เลือกได้ว่าจะยืนยันด้วย KYC หรือยืนยันผ่านแอพ Krungthai NEXT |

จากตารางจะเห็นว่าแอพที่ย้ายไปเครื่องใหม่ได้ง่ายที่สุดจะเป็น ttb touch ครับ เรากรอกแค่เลขบัตรประชาชน ส่วน SMS OTP จะส่งมาให้เองด้วยเบอร์ที่เราเคยลงทะเบียนไว้

แอพที่ยากที่สุดจะเป็น krungsri ครับที่เราจำเป็นที่จะต้องออกจากระบบในเครื่องเก่าก่อนไม่งั้นจะย้ายไปเครื่องใหม่ไม่ได้ ถ้าเกิดใครมือถือพังหรือหายออกจากระบบไม่ได้ ต้องเดินทางไปทำที่สาขาครับ

ทั้งนี้ข้อมูลในตารางเป็นข้อมูลที่ผมทดสอบเองเฉพาะแอพธนาคารที่ได้ลองย้ายนะครับ หากขาดแอพธนาคารไหนไปอาจเพราะผมไม่ได้ย้ายแอพนั้นหรืออาจะไม่ได้ใช้แอพนั้นครับ และข้อมูลที่เห็นเป็นข้อมูลที่บันทึกจากช่วงเวลาหนึ่งเท่านั้น แอพมีการอัพเดทตลอดปัจจุบันอาจจะไม่ตรงกับข้อมูลในตารางแล้ว
